## python 爬虫示例仓库
### getArticle
    1. getArticle
    
    爬虫获取文章内容(以金融之家 http://www.jrzj.com/ 为例)标题写入txt
    2.getArticleInCSV
    
    爬虫获取文章内容(以金融之家 http://www.jrzj.com/ 为例)标题，标签，日期写入csv
    3.getArticleSql
    
    爬取文章保存到数据库
